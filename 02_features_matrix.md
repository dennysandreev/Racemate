## Источники данных MVP

### RSS

Для новостей.

Сохраняем:
- заголовок;
- описание;
- источник;
- дату;
- ссылку;
- raw payload;
- AI summary.


### Jolpica

Для:
- календаря;
- гонок;
- результатов;
- standings;
- пилотов;
- команд;
- трасс.

Нужно кешировать и обрабатывать 429.

### OpenF1

Для:
- historical session data;
- race control messages после сессии;
- pit stops;
- weather;
- laps;
- session metadata.

В MVP не использовать live window.

### FastF1

Для будущей аналитики после гонки:
- графики темпа;
- сравнение напарников;
- пит-стопы;
- telemetry analysis.

Лучше вынести в Python worker.

### OpenRouter

Для:
- саммари;
- перевода;
- дневных сводок;
- предгоночных и постгоночных отчетов.

### Open-Meteo

Для погоды по координатам трассы.

### Supabase

Для:
- Auth;
- Postgres;
- RLS;
- хранения user data.

### Resend/Postmark/SendGrid

Для production email OTP через custom SMTP.
