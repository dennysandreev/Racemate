# 07. Telegram Bot Spec

## Роль

Бот нужен для:
- привязки Telegram к email-аккаунту;
- ежедневных сводок;
- напоминаний о сессиях;
- результатов;
- новостей по избранным командам/пилотам;
- прогнозов и мини-лиг.

## Команды

### /start

Если без токена:

```text
Привет! Я бот F1 Fan Hub.

Я могу присылать:
— главные новости F1;
— напоминания о сессиях;
— результаты;
— прогнозы и мини-лиги.

Чтобы подключить уведомления, войди на сайте и нажми “Подключить Telegram”.
```

Если с токеном `/start link_<token>`:
- проверить токен;
- привязать Telegram;
- ответить успехом.

### /help

```text
Команды:
/summary — главное за день
/next — ближайшая сессия F1
/settings — настройки уведомлений
/unsubscribe — отключить уведомления
```

### /summary

Отправить последний daily digest.

### /next

Отправить ближайшую сессию:
- этап;
- тип сессии;
- дата/время в timezone пользователя;
- ссылка на Race Weekend Hub.

### /settings

Показать кратко настройки и ссылку на сайт.

### /unsubscribe

Отключить Telegram-уведомления.

## Уведомления

### Daily digest

```text
🏁 Главное в F1 за сегодня

1. ...
2. ...
3. ...

Читать подробнее: <link>
```

### Race reminder

```text
⏰ Через 1 час начнется квалификация

Гран-при ...
Время: ...
Открыть уикенд: <link>
```

### Results

```text
🏁 Итоги квалификации

1. ...
2. ...
3. ...

Полные результаты: <link>
```

### Favorite team news

```text
🔔 Новость по Ferrari

<title>

<summary>

Источник: <source>
Оригинал: <link>
```

## Technical

- Все отправки через очередь.
- Логировать в `notification_logs`.
- Если пользователь заблокировал бота — `telegram_accounts.is_active=false`.
- Respect notification preferences.
- Не спамить.
